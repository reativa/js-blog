3: O que o operador OR || faz?

February 03, 2020

View this post on Instagram

Fala Reativers 🐿 o nosso 3° IGTV da série é sobre o operador OR || Podemos usar o OR para fazer a análise lógica comum 🔄, ou podemos atribuir valores padrões para as nossa variáveis, caso o valor que esteja sendo passado não exista. Sobre o valor padrão, utilizamos uma feature do ES6 para demonstrar que o OR já não é utilizado para este fim em códigos mais modernos 🕹️.

A post shared by Paulo Luan | Reativa (@reativatecnologia) on Mar 4, 2020 at 4:01pm PST

O operador || ou Logical OR encontra a primeira expressão verdadeira em seus operandos e a retorna. Isso também emprega curto-circuito para evitar trabalho desnecessário.

console.log(null || 1 || undefined) // logga 1, nem chega a executar o resto

Ele inicializava valores padrões antes do ES6 chegar. 

function logName(name) {
  var n = name || "Mark" // atribui a variável Mark caso não tenha nome
  console.log(n)
}

logName() //logs "Mark"

Exemplo no ES6:

function logName(name = 'Mark') { // atribui a variável Mark caso não tenha nome
  var n = name 
  console.log(n)
}

logName() //logs "Mark"
